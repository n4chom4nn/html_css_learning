HTML Tags Overview

This document contains an example of every HTML tag along with a brief description.

Document Metadata

<!DOCTYPE> - Defines the document type and version of HTML.

<html> - The root element of an HTML document.

<head> - Contains metadata/information for the document.

<title> - Sets the title of the document, shown in the browser's title bar or tab.

<meta> - Defines metadata about an HTML document.

<link> - Defines the relationship between the current document and an external resource.

<style> - Contains style information for a document.

<script> - Defines a client-side script.

Content Sectioning

<header> - Represents introductory content or a set of navigational links.

<nav> - Defines a set of navigation links.

<section> - Represents a standalone section of content.

<article> - Represents a self-contained composition in a document.

<aside> - Represents content indirectly related to the main content.

<footer> - Represents the footer for its nearest sectioning content or sectioning root element.

Text Content

<h1> to <h6> - Defines HTML headings.

<p> - Defines a paragraph.

<hr> - Represents a thematic break between paragraph-level elements.

<pre> - Represents preformatted text.

<blockquote> - Represents a section that is quoted from another source.

<ol> - Defines an ordered list.

<ul> - Defines an unordered list.

<li> - Defines a list item.

<dl> - Represents a description list.

<dt> - Represents a term in a description list.

<dd> - Represents the description of a term in a description list.

<figure> - Represents self-contained content, often with a caption.

<figcaption> - Represents a caption or legend for a figure.

Inline Text Semantics

<a> - Defines a hyperlink.

<em> - Represents emphasized text.

<strong> - Represents strong importance.

<small> - Represents side-comments and small print.

<s> - Represents text that is no longer correct.

<cite> - Represents the title of a work.

<q> - Represents a short inline quotation.

<abbr> - Represents an abbreviation or acronym.

<time> - Represents a specific period in time.

<code> - Represents a fragment of computer code.

<var> - Represents a variable in a mathematical expression or programming context.

<samp> - Represents sample output from a computer program.

<kbd> - Represents user input.

<sub> - Represents subscript text.

<sup> - Represents superscript text.

<i> - Represents text in an alternate voice or mood.

<b> - Represents text stylistically different from normal text.

<u> - Represents text with an unarticulated, though explicitly rendered, non-textual annotation.

<mark> - Represents highlighted text.

<ruby> - Represents ruby annotations.

<rt> - Represents the text of a ruby annotation.

<rp> - Represents parentheses around a ruby annotation.

<bdi> - Represents text that is isolated from its surrounding for bidirectional text formatting.

<bdo> - Represents the directionality of text.

<span> - Represents a generic inline container for phrasing content.

<br> - Represents a line break.

<wbr> - Represents a word break opportunity.

Image and Multimedia

<img> - Embeds an image into the document.

<audio> - Embeds sound content.

<video> - Embeds video content.

<source> - Specifies multiple media resources for media elements.

<track> - Specifies text tracks for media elements.

<embed> - Embeds external content.

<object> - Represents an external resource.

<param> - Defines parameters for an object element.

<picture> - Contains zero or more source elements and one img element.

<map> - Defines an image map.

<area> - Defines a clickable area within an image map.

Embedded Content

<iframe> - Represents a nested browsing context.

<canvas> - Used to draw graphics via scripting.

<svg> - Defines a container for SVG graphics.

<math> - Embeds a MathML formula.

Scripting

<script> - Defines a client-side script.

<noscript> - Defines alternative content for users that do not support client-side scripts.

<template> - Holds client-side content that will not be rendered when the page loads.

<slot> - Acts as a placeholder inside a web component.

Forms

<form> - Represents a document section containing interactive controls for submitting information.

<input> - Represents a typed data field.

<textarea> - Represents a multi-line plain-text editing control.

<button> - Represents a clickable button.

<select> - Represents a control that provides a menu of options.

<optgroup> - Groups options within a select element.

<option> - Represents an option in a select element.

<label> - Represents a caption for an item in a user interface.

<fieldset> - Groups related elements in a form.

<legend> - Represents a caption for the content of its parent fieldset.

<datalist> - Contains a set of option elements that represent the permissible or suggested options available to other controls.

<output> - Represents the result of a calculation or user action.

<progress> - Represents the completion progress of a task.

<meter> - Represents a scalar measurement within a known range.

Table Content

<table> - Represents tabular data.

<caption> - Represents the title of a table.

<colgroup> - Defines a group of columns within a table.

<col> - Specifies column properties for each column within a colgroup element.

<tbody> - Groups the body content in a table.

<thead> - Groups the header content in a table.

<tfoot> - Groups the footer content in a table.

<tr> - Represents a row of cells in a table.

<td> - Represents a data cell in a table.

<th> - Represents a header cell in a table.

Interactive Elements

<details> - Represents a disclosure widget from which the user can obtain additional information.

<summary> - Represents a summary, caption, or legend for a details element.

<dialog> - Represents a dialog box or other interactive component.

End of HTML Tags Overview
